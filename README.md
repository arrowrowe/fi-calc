# fi-calc

---

金融计算. 支持
- 还款计划生成, 包括: 按月付息到期还本, 等额本金, 等额本息.
- 自定义取整方式, 包括: 四舍五入, 向上取整, 向下取整.
- 自定义金融字符串的格式化, 包括: 是否按千分加逗号, 前缀, 后缀.

## Installation

```
$ spm i --save fi-calc
```

```javascript
var fiCalc = require('fi-calc');
```
